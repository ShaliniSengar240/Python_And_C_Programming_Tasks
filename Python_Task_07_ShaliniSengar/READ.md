## 🎯 Objective

The objective of this project is to understand and implement Object-Oriented Programming (OOP) concepts in Python by building a menu-driven Employee Management System. This application helps in managing employee records efficiently while demonstrating the practical use of classes, objects, file handling, and exception handling.

---

## 🚀 Features

* Add new employees.
* View all employee records in tabular format.
* Search employees by ID or Name.
* Update employee details.
* Delete employee records.
* Display salary statistics.
* Export employee data to a file.
* Read and display saved records.
* Menu-driven user interface.
* Uses Object-Oriented Programming concepts.

---

## 🛠️ Modules Implemented

### Part A: Employee Class

Created an `Employee` class containing:

* Employee ID
* Employee Name
* Department
* Designation
* Salary

Used the `__init__()` constructor to initialize employee details.

---

### Part B: Add Employee

Allows users to:

* Add employee details.
* Store employee objects in a list.
* Display a success message after adding records.

---

### Part C: View Employees

Displays all employee records in a clean tabular format.

Example:

```text id="x7k1n2"
------------------------------------------------------------
ID     Name       Department    Designation    Salary
------------------------------------------------------------
101    Soham      IT            Developer      50000
102    Rahul      HR            Manager        65000
```

---

### Part D: Search Employee

Search employees using:

* Employee ID
* Employee Name

Displays employee details if found; otherwise shows:

```text id="m4a8q1"
Employee Not Found
```

---

### Part E: Update Employee

Allows updating:

* Department
* Designation
* Salary

Displays a confirmation message after successful update.

---

### Part F: Delete Employee

Deletes employee records using Employee ID.

Displays:

```text id="z3v8r6"
Employee Deleted Successfully
```

---

### Part G: Salary Statistics

Displays:

* Highest Salary
* Lowest Salary
* Average Salary
* Total Number of Employees

---

### Part H: Export Data

Exports employee records to:

* `employees.csv`
  **or**
* `employees.txt`

Reads the saved file and displays all employee records.
