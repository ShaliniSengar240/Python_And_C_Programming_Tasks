# Student Cyber Security Management System

## 🎯 Project Objective

The purpose of this project is to combine all concepts learned during the internship, including:

- Variables and Data Types
- Input and Output Operations
- Conditional Statements
- Loops
- Functions
- Arrays/Lists
- File Handling
- Searching Techniques
- Basic Cyber Security Concepts
- Problem Solving

This project is developed in both **Python** and **C Programming** and demonstrates a practical student management application integrated with cybersecurity features.

---

# 📌 Project Title

## Student Cyber Security Management System

A menu-driven application that manages student records and performs basic cybersecurity assessments.

---

# ✨ Features

## Student Management

### 1. Add Student
Stores:
- Student Name
- Student ID
- Branch
- Email Address

### 2. View Students
Displays all stored student records.

### 3. Search Student
Search student records using:
- Student Name
- Student ID

### 4. Delete Student
Deletes student records and displays a confirmation message.

---

## 🔒 Security Assessment Module

The system evaluates cybersecurity awareness using:

- MFA (Multi-Factor Authentication) Status
- Password Length
- System Update Status
- Antivirus Installation Status

### Security Categories

- Excellent
- Good
- Moderate
- Poor

### Sample Output

```
Security Score: 85/100

Status:
Good Security Practices
```

---

## 📊 Report Generation

Generates a summary report containing:

- Total Students
- Individual Security Scores
- Average Security Score
- Students with Poor Security Ratings

---

## 📁 File Handling

The project stores data inside:

```
students.txt
```

Operations performed:

- Save Records
- Read Records
- Update Records
- Delete Records

---

# 🛡 Cyber Security Features Implemented

### Password Strength Checker
Checks whether a password is weak, moderate, or strong based on:

- Length
- Numbers
- Special Characters
- Uppercase Letters

### Username Generator
Automatically generates usernames from student details.

### Login Validation
Validates user credentials before allowing access.
