## Part D: Cyber Security Log Analyzer

### Files
- log_analyzer.py
- log_analyzer.c

### Description
This program analyzes login attempt records and displays:

- Total Login Attempts
- Successful Logins
- Failed Logins

### Sample Output
```
Total Attempts: 6
Successful Logins: 3
Failed Logins: 3
```

### Importance in Cybersecurity
Monitoring failed login attempts is important because it helps identify:

- Unauthorized access attempts
- Brute-force attacks
- Suspicious user activity
- Potential security threats

Early detection helps improve system security and prevent breaches.

## Part E: Blacklisted IP Checker

### Files
- blacklist_checker.py
- blacklist_checker.c

### Description
This program checks whether an entered IP address exists in a predefined blacklist.

Blacklisted IPs:
