# Python and C Programming

## 📌 Task 03: Loops, Patterns & Basic Automation

### 🎯 Objective
The purpose of this task is to strengthen understanding of loops, iteration, problem-solving, and basic automation concepts. The programs were implemented in both Python and C Programming.

---

# Part A: Multiplication Table Generator

### Files
- multiplication_table.py
- multiplication_table.c

### Description
This program accepts a number from the user and displays its multiplication table from 1 to 10 using loops.

### Example Output
```
Enter a number: 7

7 x 1 = 7
7 x 2 = 14
7 x 3 = 21
...
7 x 10 = 70
```

---

# Part B: Number Analysis Tool

### Files
- number_analysis.py
- number_analysis.c

### Description
This program accepts a number **N** from the user and calculates:

- Sum of numbers from 1 to N
- Count of even numbers
- Count of odd numbers

### Example Output
```
Enter a number: 10

Sum = 55
Even Numbers = 5
Odd Numbers = 5
```

# Part D: Password Attempt Simulator

### Files
- password_attempt.py
- password_attempt.c

### Description
This program simulates a basic login system.

### Features
- Stores a predefined password
- Allows a maximum of 3 attempts
- Displays:
  - Access Granted
  - Account Locked (after 3 failed attempts)

### Example Output
```
Enter Password: admin123

Access Granted
```

or

```
Enter Password: test
Enter Password: demo
Enter Password: user

Account Locked
```
